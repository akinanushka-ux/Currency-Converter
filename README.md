# Currency Converter Project

**Автор:** [Арина Дерескова]

### Описание
Приложение для конвертации валют с использованием актуальных данных и сохранением истории запросов.

### Как получить API-ключ
1. Перейдите на сайт [exchangerate-api.com](https://exchangerate-api.com).
2. Введите свой Email и нажмите "Get Free Key".
3. Подтвердите почту и скопируйте ключ из личного кабинета.
4. Вставьте ключ в переменную `API_KEY` в файле `main.py`.

### Использование
1. Запустите программу: `python main.py`.
2. Введите сумму (число больше 0).
3. Выберите валюты из списка.
4. Нажмите "Конвертировать".
5. Результат появится в окне и сохранится в таблицу внизу.
